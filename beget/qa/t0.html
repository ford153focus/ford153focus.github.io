sites second site in same folder Здравствуйте. Вы можете создать поддомен на https://cp.beget.com/subdomains (если речь идёт о поддомене) или прилинковать к нужному сайту еще один домен на https://cp.beget.com/sites (если речь идёт о домене). После этого вам нужно будет добавить в файл .htaccess, который находится в директории public_html нужного сайта, строки следующего вида

``` php
RewriteEngine On
RewriteCond %{HTTP_HOST} ^subdomain.example.com$ [NC]
RewriteCond %{REQUEST_URI} !^/subdomain/ [NC]
RewriteRule (.*) /subdomain/$1 [L]
```

/subdomain/ заменяем на имя директории, в которую мы будем заливать "подсайт", subdomain.example.com меняем на нужный домен/поддомен
